Kebijakan Privasi

Terakhir diperbarui: 6 Oktober 2025

Aplikasi Quiz Kosakata Bahasa Korea menghormati privasi pengguna dan berkomitmen untuk melindungi data pribadi Anda. Halaman ini menjelaskan bagaimana kami menangani informasi yang terkait dengan penggunaan aplikasi ini.

1. Informasi yang Kami Kumpulkan

Aplikasi ini tidak mengumpulkan data pribadi apa pun dari pengguna. Kami tidak meminta nama, alamat email, nomor telepon, atau informasi pribadi lainnya.

2. Data yang Disimpan Secara Lokal

Beberapa data seperti hasil kuis, skor, atau sesi terakhir dapat disimpan di perangkat Anda menggunakan local storage. Data ini hanya tersimpan di perangkat Anda sendiri dan tidak dikirim ke server mana pun.

3. Koneksi Internet

Aplikasi ini mungkin menggunakan koneksi internet hanya untuk memuat data kosakata dari sumber daring (misalnya GitHub atau server pembelajaran). Kami tidak melakukan pelacakan atau pengumpulan data dari koneksi tersebut.

4. Izin Aplikasi

Aplikasi ini tidak memerlukan izin akses khusus seperti kamera, lokasi, kontak, atau penyimpanan eksternal.

5. Keamanan

Kami menjaga agar aplikasi tetap aman dan tidak mengumpulkan data apa pun. Karena tidak ada data pribadi yang dikumpulkan, risiko penyalahgunaan data sangat kecil. Namun, pengguna disarankan untuk selalu memperbarui aplikasi ke versi terbaru.

6. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan berlaku segera setelah versi baru diterbitkan di Google Play Store.
